¿Qué es CallBack?

Un callback es una función que se pasa como argumento a otra función para ser ejecutada más tarde.

¿Para qué Funciona Callback?

Un callback funciona para ejecutar una función una vez se haya completado otra, permitiendo controlar el orden de ejecución en operaciones asincrónicas como lecturas de archivos, solicitudes a servidores o eventos del usuario.
